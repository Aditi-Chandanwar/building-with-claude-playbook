# Building with Claude Playbook

## Purpose

This repository documents my learning journey through Anthropic's **Building with Claude** course.

Rather than serving as a transcript of the course, this playbook captures my understanding, experiments, reflections, and practical applications as I learn to build AI systems.

---

## Learning Philosophy

I believe the best way to learn AI engineering is to:

- Understand concepts instead of memorizing them.
- Explain ideas in my own words.
- Build small experiments.
- Apply every important concept to a real project.
- Reflect on what worked, what didn't, and why.

---

## Repository Structure

| Folder          | Purpose                                     |
| --------------- | ------------------------------------------- |
| `modules/`      | Notes and summaries written in my own words |
| `experiments/`  | Small practical implementations             |
| `applications/` | Applying concepts to real-world projects    |
| `prompts/`      | Reusable prompt patterns                    |
| `resources/`    | References, glossary, and useful links      |
